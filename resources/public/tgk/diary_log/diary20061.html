2006.12.09(Sat)
大学に入ってからの自分のこと

まず，おおよそやりこみとは縁を切ってる感じです．最近買ったFF5Aで低レベルABP0をやったりはしましたが，なんていうかもう制限プレイに面白さを感じなくなりました．とは言え東方でドットイートや低速・高速封印に挑戦したりする辺り，まだ意欲が枯れ果てた訳ではなさそう？　しかし自分の人生でのトップクラスの趣味ではなくなりました．それは間違いないです．

数学と物理への意欲は衰えるどころか溢れる一方です．複素関数論とか楽しくて仕方ありません．ε-δ論法が3年の歳月をかけてようやく理解出来たのも嬉しい限りです．（高校一年の数列のときに聞いて以来の疑問だったから，多分3年くらい）
けど全微分の定義の意味がよく分かってなかったり，電磁気学はサパーリで力学も換算質量とかの話がまだよく分かってないとか言う辺り，やはりいい加減なものなのかもしれません(^^;

改めて目覚めたことはプログラミングです．C++最高．
今はゲームを作りまくってます．まだ一つも完成させてませんが，年内に一つ完成させる予定です．でも知人以外には公開しませんよｗ　公開する作品は少なくとも来年の夏頃まで待って下さい．
とは言え Visual Studio をまったく使えてなかったり（開発環境は BCC + CPad + BCC Developer の複合技だぞｗ） ，Windows プログラムをいい感じに放棄していたり，DirectX がサパーリだったりで，やはりなんだかんだ言っていい加減なのは相変わらずの自分です orz
でも C++ は同期の中で一番使いこなせてる自信がある！（ヒント：私以外誰もメインで使ってない orz）

後は，バイトを始めて溜まったお金で高校の頃は買えなかったようなもの（グッズとかDVDとか）を色々買ったり，日本全国あちこち旅行してみたり（東京以北は未踏），ゲーセンでやたら人間関係が広がったり（ゲーセンにいられるのはバイトのお陰），と，そんな感じです．かなり楽しい学生生活を送れていると思ってます．あと2年ちょっとは親元にいられるから，もう少しこの贅沢な暮らしを堪能しておきたいです(^^;

というような様子で，私は今生きています．結構幸せです(´∀｀)
