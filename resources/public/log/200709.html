xyzzy 設定メモ

以下， ~/ は xyzzy.exe があるフォルダとする
設定は ~/.xyzzy もしくは ~/site-lisp/siteinit.l に lisp を用いて記述する
.xyzzy や siteinit.l というファイル自体は自作する
ユーザ環境変数（システム環境変数でも可）に「変数：XYZZYHOME　値：.xyzzyがあるディレクトリ」というものを付け加える
（例：XYZZYHOME C:\Program Files\xyzzy\）
.xyzzy は起動時に毎回読み込まれる
M-x byte-compile-file で Enter を押し siteinit.l を指定するとバイトコンパイルが完了し ~/site-lisp/siteinit.lc が出来る
siteinit.l または siteinit.lc の設定はダンプファイルの中に反映される
ダンプファイルは Windows XP なら ~/xyzzy.wxp となり，Windows 2000 なら ~/xyzzy.w2k となる（※他には xyzzy.w98 xyzzy.wnt xyzzy.wme）
ダンプファイルが存在しない状態で xyzzy を起動するとダンプファイルは作成される
〜役立ち設定〜
Windows のショートカットキーを使う (load-library "gates")　（※~/lisp/Gates.l を読み込むという意味）
デフォルトのディレクトリを指定 (set-default-directory "C:/doc/text/")　（※\ではなく/であることに注意）
テキストの折り返し設定
折り返さない (set-buffer-fold-type-none)
指定位置で折り返す (set-buffer-fold-type-column)
ウィンドウ端で折り返す (set-buffer-fold-type-window)

相棒の写真

東プレの Realforce91U (17,600円) も PFU の Happy Hacking Lite2 (6,000円) も所持し，更に加えて6つものキーボード（多分全部で10,000円ちょい）を所持し，更に更に今まで買って人に譲った分も含めるともうほんと5万円くらい行くんじゃね？てな具合にキーボードバカのわたくし．そして実はこの男，マウスやトラックボールというポインティングデバイスの方も結構な品数（それも3,000円は軽く越えるようなモノばかり）を所持しているので，冗談抜きで今まで入力デバイスにかけたお金で海外旅行に行けそうです．

と，そんな話はさておくとして（悲しいので），このように約3年間キーボードにこだわり続けてきた結果，ようやく最良のキーボードが決まりそうな感じです．


それがこの子．Logitech の1,200円ちょいのキーボードです．もうちょっと大きい写真はコチラ


フルアーマー装備パームレストを装着した状態．

皮肉なものです．使い易いキーボードを求め東奔西走し（マジ），高いお金をつぎ込んだ結果が1,200円の品なのです．

しかし，コイツの軽さ（本体の軽さと打鍵の軽さ両方とも）には素晴らしいモノがあります．何より水をこぼしても大丈夫というのが良い．

そして，安いキーボードならではというか，キー配列が日本の業界標準（と言ってもいいだろう）なのです．これは何がいいかって，誰でも違和感なく使えるということです．以前サモが我が家で Happy Hacking を使っていて「半角/全角キーってどこ？」と聞いてきたことはやはり意識すべきなのです．メッセとかやってる人ならエスケープ誤爆の恐ろしさが分かるはず．分かりやすく携帯に例えるなら，「し」を打とうとして誤って電源ボタンを2回押してしまい「アッー！」てな感じです．

まぁもちろん欠点がないわけではありません．
まずフルキーボードの最大の欠点，「横幅が長い」すなわち「マウスが遠い」．
そして「ホットキーのような便利機能が皆無」．
で，これはこいつ固有の欠点とは言えない気もしますが，「Realforce よりは打鍵が重い」．Realforce が軽すぎとも言う．

でも何故か気付くとコイツを使いたくなるんですよ．Realforce 使ってても，やっぱりコイツの方がしっくり来るなぁ…なんて思ってキーボードを替えるわけですよ．これはもう運命の赤い糸みたいなもんですね．一応「最高のキーボードを探す旅」は続けますが，僕はこの相棒ともう3年くらいを共にしようと思います．

ちなみにコイツの型番は Logitech (旧Logicool) の iK-20** （** には BK や WH などカラーを表す二文字が入る）というものなんですが，もう Logitech のサイトには載っていないし，アマゾンでも在庫切れとなっています．恐らく生産中止なのだと思います．だから，もし店頭などに売っているのを見つけたら買っておいて損はないと思います．ちなみにもちろん私は予備を1つ買いましたｗｗｗｗ
