継承(C++)

一回継承して作られた新しいクラスを再び継承するとき，仮想関数かどうかが分からなくなるという問題があるんじゃないかと思います．

つまりこういうことです．

#include <iostream>
using namespace std;

class Base {
public:
　virtual void func() { cout << "Base::func()" << endl; }
};

class Derive : public Base {
public:
　void func() { cout << "Derive::func()" << endl; } //virtual と書かない
};

class Derive2 : public Derive {
public:
　void func() { cout << "Derive2::func()" << endl; }
};

int main()
{
　Base *pBase1 = new Base;
　Base *pBase2 = new Derive;
　Base *pBase3 = new Derive2;
　
　pBase1->func();
　pBase2->func();
　pBase3->func();
　
　delete pBase3;
　delete pBase2;
　delete pBase1;
}

-- 実行結果 --
Base::func()
Derive::func()
Derive2::func()

Derive はメソッド func を virtual に指定していません．あくまでも Derive の親である Base が func を virtual に指定してあるだけです．ここで Derive しか見えない（Base が見えない）人が居たとして，次のような処理を書いたとしたら…

int main()
{
　Derive *pDerive1 = new Derive;
　Derive *pDerive2 = new Derive2;
　
　pDerive1->func();
　pDerive2->func();
　
　delete pDerive2;
　delete pDerive1;
}

-- 実行結果 --
Derive::func()
Derive2::func()　←これがおかしいと思ってしまわないか

あくまでも Derive の定義部分しか見えていないわけですから，virtual というキーワードを見つけられず，メソッド func は Derive2 で定義してもオーバーライドになるとは考えられないはずです．

-- 実行結果（願望） --
Derive::func()
Derive::func()
となって欲しいのに，オーバーライドの機能が勝手に使われてしまって…というわけ

もちろん，これは現実問題としてほぼありえないことだとは思います．継承元のクラスが見えないとか，わざわざ基底クラスのポインタに代入しているのにオーバーライドの機能を使おうとしないとか．実際は，Derive を更に継承して使って欲しいと思うなら Derive の func 定義部分に virtual を明示的に書いておくべきだし．しかし，言語仕様的にこれはどうなんよ？と言う感じ．

入門書だと，継承の例って必ず一回しか継承してないんですよね．だから継承を二回するということ自体全てにおいて初体験で，色々戸惑ってます．

らき☆すたの聖地巡礼で何やら穏やかでない状況が…

聖地巡礼の「らき☆すた」ファンが地元住民に不審者扱いされる（痛いニュース(ノ∀｀)）

そしてこのきっかけとなる記事を書いたブログ（オタクが集まる鷲宮神社（我ら久喜市民のHP ））は，当然というかまぁ大炎上しています．

まぁ双方とも言い分はあるでしょうが，自分の意見としては，まずヲタ擁護の観点から地元住民に言わせて貰いたいのは「韓流ブームで韓国に大挙して押し寄せたオバハン達と別に変わりないやん？」「不審者が増えているのだろうが，“治安は悪くなってない”」ということ．そして地元住民擁護の観点からヲタの方々に言わせて貰いたいのは「怖がっている人がいる事実がある以上どこかで何らかの自主規制をするべき」ということ．

自分も雛見沢訪問と言う名の白川郷観光を去年してきました（そのときの記事はコチラ）．通称“嘘だ坂”で「嘘だッ！！！！」と叫んだり，今思えば一般観光客に迷惑なことをしたんじゃないかなぁ…　と思います．ていうか木刀を持って悟史バットとか言って闊歩してたしｗ

でも決定的に違うのが，白川郷はそもそもそこが観光地なんですよね．知らない人が大量に来ることが通常という．それに対して今回のらき☆すたの件は，何の変哲も無いごく普通の住宅街．そりゃ毎日毎日カメラ携えたジーパンとTシャツとリュックサックがグループ形成して代わる代わる来れば，怖がるのも無理はないと思いますよ．何せ一般ピーポーからすれば我々は異種民族と言っても過言じゃないんですから．

とは言っても，やはり聖地巡礼の比類なき楽しさは認めます．こればっかりはやったことのある人じゃないと絶対分からない．目的地近辺に到着し，「ここだ！」と思う位置を特定して，そしてカメラのファインダーを覗きつつ（ってもうデジカメが主流の現代じゃこの表現も古いかなぁ…）劇中の風景を再現出来たときの楽しさというか達成感．これはほんと何とも言えないものです．アニメでもドラマでも映画でも，それこそ写真作品というものでも，とにかく映像文化全てに共通することだと思うよ．

結局どっちつかずの意見しか言えないんですが，どっちかに偏った意見を出す必要もないんじゃね？というのが今回の私の意見でもあって(^^;　住民側の言ってることもヲタ側の言ってる事も分かるんだけど，そのどっちも正しくて，間違ってないんだけど，だから相手側が間違っているんだ，と言うことも出来ないわけで．ヌルい回答だけど「お互いが譲り合って丁度いい折衷案を出せたらいいよね」ってことで，ひとつ．

まぁ，お互い冷静になろうよ．繰り返すけど，治安は悪くなってないよｗ　あと，田んぼの撮影やってるところ（痛いニュースの2chでの反応の475の画像）で，さすがにこれだと不審者ですよｗ
